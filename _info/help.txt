### Check Status ###
[Status]: Report all status of each equipment
[Mount]: Report status of Mount
[Camera]: Report status of Camera
[Focus]: Report status of Focuser

### Basic Command ###
[clear]: Clear terminal
[quit]: End the program

### Mount Command ###
[goto rd (ra) (dec)]: Move telescope to Equitorial coordinate 
[goto aa (alt) (az)]: Move telescope to Horizontal coordinate 
[sync rd (ra) (dec)]: Synchronize telescope to Equitorial coordinate 
[sync aa (alt) (az)]: Synchronize telescope to Horizontal coordinate
* (HH:MM:SS.SS) format for DEC
* (DD:MM:SS.SS) format for RA, ALT, AZ

[tracking (on/off)]: On/Off the sidereal tracking system

[park]: park telescope
[unpark]: unpark telescope

[ra  (amp) (unit)]: Move telescope in RA  axis
[dec (amp) (unit)]: Move telescope in DEC axis
[alt (amp) (unit)]: Move telescope in ALT axis
[az  (amp) (unit)]: Move telescope in AZ  axis
* Units: d(degree), h(hour), m(arcminute), s(arcsecond)
* h for DEC
* d for RA, ALT, AZ
* m & s are used for both arcsecond & arcminute(RA, ALT, AZ) and second & minute (DEC)

### Camera Command ###
[set cooler (temperature)]: Set cooler temperature
[exposure (filename) (exposuretime) (repeat)]: Exposure light image

[flat (filename) (exposuretime) (repeat)]: Exposure flat image
[dark (filename) (exposuretime) (repeat)]: Exposure dark image
[bias (filename) (repeat)]: Exposure bias image
* You can only type the [exposure] [flat] [dark] [bias]

[binning setting]: Set the X binning and Y binning
* Default is 1.

### Focuser Command ###
[focus (amplitude)]: change focus position into (current focus) + (amplitude)
[focus auto]: autometically find focus

### Extra Command ###
[query (filename)]: query image -> sync to Equitorial coordinate
